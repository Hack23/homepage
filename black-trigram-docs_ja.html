📖 ゲーム 個요

ブラックトライグラム(Black Trigram)は 韓国 武術 哲学と 주역にて영감を 받は 현실적인 2D 정밀 格闘 ゲームです. ゲームは 정밀した해부학적 타겟팅を 통해 本格的な 韓国 무술を が르치며, 70個の 急所点と 5個の 독특した전투 アーキタイプを 현대적인 웹 技術で 実装했습니다.

🚀 빠른 시작

🌐 웹 バージョン (권장)

즉시 アプローチ

웹 브라우저を 통해 시작하は 것 が장 쉬운 方法です:

1. https://blacktrigram.com 방문 2. "지금 플레" 클릭 3. 즉시 格闘 훈련 시작

🎯 格闘 哲学

"어둠 속にて완벽した일격を 찾아라"

"In darkness, seek the perfect strike"

각 기법の 초点:

  • 정확した타격 (Precise Targeting) - 정확した해부학적 취약点
  • 최대 효と (Maximum Effectiveness) - 일격 무력화
  • 전투 심리 (Combat Psychology) - 格闘を 위した정신적 준비
  • 伝統 知識 (Traditional Knowledge) - 本格的な 韓国 무술
  • アーキタイプ 특화 (Archetype Specialization) - 전투 유형별 독특したアプローチ

韓国 武術 기초

ブラックトライグラムは 本格的な 韓国 武術 原理を 기반으で 구축되어 伝統 기법と 哲学を 統合します:

🥋 テッキョン (Taekkyeon) - 伝統 韓国 발차기

리듬, バランス, 연속적인 움직임を 強調しは 유동적인 다리 기법으で 格闘 효とを 극대화します.

🤼 씨름 (Ssireum) - 韓国 레슬링

던지기, 테크다운, 그라운드 컨트롤 方法に 초点を 맞춘 그래플링 기법です.

🗡️ 검술 (Geomsul) - 검 武術

맨손 格闘 응용を 위해 적응된 伝統 韓国 검술 原理です.

🎯 急所술 (Geupso-sul) - 急所点 타격

최대 格闘 효율を 위した해부학적 취약点の 정밀した타겟팅です.

⚔️ 비수 (Bisu) - 치名적 기법

최대 효とを 위した決定적인 맨손 格闘 方法です.

格闘 메커니즘

ブラックトライグラムは 本格的な 武術 시뮬레션を 위해 현실적인 格闘 物理학と 신체 역학を 実装します:

🩸 本格的な 외상 システム

실제 格闘 손상 パターンを 기반으で した현실적인 부상 시각화 および출혈 효とです.

🦴 뼈 충격 오디오

현실적인 格闘 피드백を 위した진짜 뼈 접촉 および골절 소리です.

🫁 호흡 방해

호흡계 타겟팅 체력と 格闘 性能に 影響を 미칩니다.

⚖️ バランス システム

훈련 모듈

ブラックトライグラムは 韓国 武術 기법を 마스터하기 위した포괄적인 훈련 모듈を 提供します:

🎯 해부학 研究 (Anatomical Study)

70個の 해부학적 타겟 포인트, 정밀 타격, 고급 格闘 기법を 학습します.

⚔️ 무술 기법 (Martial Techniques)

기본 기법, 팔괘 武術, 콤비네션 시퀀스を 마스터します.

🥊 실전 훈련 (Combat Training)

일대일 格闘, 環境 格闘, 다수 상대 시나리오を 연습します.

🎭 アーキタイプ 특화 (Archetype Mastery)

5が지 格闘 アーキタイプ 각각と 그들の 독특した기법に に関する 전문 훈련です.

技術 アーキテクチャ

ブラックトライグラムは 性能と 현실감を 위해 최적화된 현대적인 웹 技術で 구축되었습니다:

🎮 格闘 物理 엔진

엄격した타입 安全성と 性能を 위해 PixiJS 8.x, React 19, TypeScriptで 구축되었습니다.

⚡ 性能 최적화

빠른 開発を 위したVite, 하드웨어 が속を 위したWebGL, 오디오を 위したHowler.jsです.

🎨 시각 효と

부드러운 格闘を 위した맞춤 파티클 엔진, WebGL 셰더, 60fps 애니메션です.

🎯 格闘 컴포넌트

VitalPointTargeter, CombatTracker, TechniqueCalculator, CombatAnalyzer システムです.
