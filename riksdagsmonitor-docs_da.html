Projektoversigt

Riksdagsmonitor er en open source svensk parlamentsintelligensplatform der overvåger politisk aktivitet med systematisk transparens. Bygget på den afprøvede Citizen Intelligence Agency (CIA) platform giver den 50+ år af historiske data, realtidsanalyse af Riksdag-aktiviteter og understøtter 14 sprog. Denne dokumentationsportal dækker den komplette arkitektur, sikkerhedsdesign, testrammer og ISMS-overensstemmelsesdokumentation.

📖 API JavaDocs 📈 Kodedækning 📚 DeepWiki 📂 GitHub

Nøglefunktioner

Parlamentarisk overvågning

Realtidssporing af svensk Riksdags aktivitet inkl. parlamentsmedlemsprofiler, stemmeregistre, udvalgsarbejde og interpellationer.

50+ år historiske data

Omfattende arkiv med over 50 år af svensk parlamentarisk aktivitet med trendanalysemuligheder.

14-sprogsupport

Fuld flersproget platform der gør svenske parlamentariske data tilgængelige for internationale målgrupper.

Politisk analyse

Avanceret adfærdsanalyse, gennemsigtighedsmålinger og evidensbaseret ansvarlighedssporing.

Datakvalitet

Automatiseret validering med digitale signaturer, hentet fra officielle Riksdag åbne data-API'er.

99.998% tilgængelighed (mål)

Højtilgængelighedsarkitektur med omfattende planlægning af forretningskontinuitet og katastrofegendannelsesstrategier.
