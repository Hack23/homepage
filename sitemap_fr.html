🚀 Projets

Projets de sécurité open source démontrant une implémentation pratique de la sécurité. Des plateformes de transparence politique aux outils de conformité et à la préservation culturelle par le jeu.

  • Portfolio de Projets
  • Black Trigram
    • AperçuSimulateur arts martiaux coréens: tradition Taekgyeon, Hapkido, gameplay authentique avec valeur éducative
    • Fonctionnalités70 points vitaux anatomiques, 5 archétypes combattants (Musa, Amsalja, Hacker, Intelligence, Crime Organisé), physique combat réaliste
    • Documentation
  • Citizen Intelligence Agency
    • AperçuPlateforme OSINT parlementaire suédoise: suivi activités politiques, métriques transparence, outil responsabilité démocratique
    • FonctionnalitésTableau de bord politique interactif, classements politiciens, analyse modèles votes, transparence données gouvernement suédois
    • DocumentationArchitecture technique Java 25/Spring Framework: PostgreSQL, pipeline données, intégration données parlementaires suédoises
  • CIA Compliance Manager
    • Aperçu
    • Fonctionnalités
    • Documentation
  • European Parliament MCP Server
    • Aperçu Serveur MCP fournissant aux assistants IA un accès structuré aux jeux de données ouverts du Parlement européen : députés, séances plénières, commissions, documents et résultats de vote
    • Fonctionnalités 10 outils MCP incluant get_meps, get_voting_records, analyze_voting_patterns, track_legislation avec sécurité de typage TypeScript, SLSA niveau 3 et plus de 80 % de couverture de tests
    • Documentation Documentation d'architecture technique pour le serveur MCP TypeScript/Node.js avec documentation ISMS, modèle de menaces, évaluation CRA et référence d'API
  • Riksdagsmonitor
    • Aperçu Plateforme d'intelligence sur le Parlement suédois surveillant l'activité politique avec une transparence systématique en utilisant plus de 50 ans de données historiques du Riksdag
    • Fonctionnalités Surveillance parlementaire, analyse des résultats de vote, suivi des commissions et analyses politiques en 14 langues avec un objectif de disponibilité de 99,998 %
    • Documentation Documentation d'architecture technique pour la plateforme de suivi du Parlement suédois avec documentation ISMS complète, architecture de sécurité et cadres de conformité
  • EU Parliament Monitor
    • Aperçu Plateforme d'intelligence sur le Parlement européen surveillant l'activité politique avec une transparence systématique, incluant les députés, séances plénières, commissions et documents législatifs
    • Fonctionnalités Suivi des députés, suivi des séances plénières, analyse des commissions, recherche de documents législatifs et questions parlementaires avec traitement des données de fonctionnaires publics conforme au RGPD
    • Documentation Documentation d'architecture technique pour la plateforme de surveillance du Parlement européen avec documentation ISMS complète, modèle de menaces, évaluation CRA et architecture de sécurité
