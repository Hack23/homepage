🚀 Open Source Projekter

Demonstrerer sikkerhedsbest practices gennem produktionsapplikationer

Hos Hack23 taler vi ikke bare om sikkerhed—vi beviser det gennem open source-projekter, der viser virkelig implementering af sikkerhedskontroller, compliance-rammer og DevSecOps-praksis. Hvert projekt opnår SLSA Level 3 supply chain-sikkerhed og opretholder aktiv sikkerhedsovervågning.

🌟 Udvalgte Projekter

🥋 Black Trigram (흑괘)

Flagskib Uddannelsesmæssigt Spilprojekt

Realistisk 3D præcisionskamp-simulator inspireret af traditionelle koreanske kampsportsarter. Indeholder 70 anatomiske vitale punkter, 5 distinkte kæmper-arketyper og autentiske kampteknikker med uddannelsesmæssig værdi og fokus på kulturel bevaring.

Nøglefunktioner:

  • 70 Anatomiske Vitale Punkter
  • 5 Unikke Kamparketype
  • Koreanske Kampsportsteknikker
  • Fysikbaseret Kamp
  • Uddannelsesmæssigt Gameplay
  • Kulturel Autenticitet

🔐 CIA Compliance Manager

Enterprise Sikkerhedsvurderingsplatform

Omfattende sikkerhedsvurderingsplatform til evaluering af Fortrolighed, Integritet og Tilgængelighed (CIA Triad) med forretningspåvirkningsanalyse og compliance-mapping til NIST, ISO 27001, GDPR, HIPAA, SOC2 og CRA-rammer.

Nøglefunktioner:

  • CIA-triade-vurdering
  • Forretningspåvirkningsanalyse
  • Multi-framework Mapping
  • Trusselmodellering (STRIDE)
  • Bevisindsamling
  • Compliance-rapportering

🔍 Citizen Intelligence Agency

Politisk Gennemsigtighedsplatform

Open-source intelligence (OSINT) platform til politisk gennemsigtighed i Sverige. Overvåger parlamentarisk aktivitet, stemmeregistre og politisk adfærd med datadrevne indsigter, analyse-dashboards og ansvarlighedsmålinger.

Nøglefunktioner:

  • Parlamentsovervågning
  • Stemmemønsteranalyse
  • Politisk Analyse
  • Datavisualisering
  • Ansvarlighedsmålinger
  • Integration af Åbne Data

🏛️ EU Parliament Monitor

European Parliament Intelligence Platform

Open-source European Parliament Intelligence Platform monitoring political activity at the EU level with systematic transparency. Comprehensive tracking of MEPs, plenary sessions, committees, legislative documents, and voting records.

Key Features:

  • MEP Monitoring
  • Plenary Session Tracking
  • Committee Monitoring
  • Legislative Document Search
  • Voting Records Analysis
  • MCP Server AI Integration

🏛️ Parlamentariske Transparensværktøjer

🏛️ European Parliament MCP Server

AI-Drevet Parlamentarisk Dataadgang

Model Context Protocol (MCP) server der giver AI-assistenter struktureret adgang til Europa-Parlamentets åbne datasæt. Giver AI-værktøjer mulighed for at forespørge MEP-data, plenarsamlinger, udvalg, afstemninger og lovgivningsdokumenter i realtid.

Nøglefunktioner:

  • MCP-protokolintegration
  • MEP-dataadgang
  • Forespørgsler om plenarsamlinger
  • Udvalgsinformation
  • Søgning i lovgivningsdokumenter
  • EU-parlamentsdata i realtid

🗳️ Riksdagsmonitor

Efterretningsplatform for Sveriges Riksdag

Swedish Parliament monitoring platform der sporer lovgivere, afstemningsoptegnelser og parlamentarisk aktivitet. Giver datadrevne indsigter i svenske politiske processer med omfattende analyser.

Nøglefunktioner:

  • Overvågning af Sveriges Riksdag
  • Lovgiversporing
  • Analyse af afstemningsoptegnelser
  • Politisk analyse
  • Ansvarsmetrikker
  • Integration med åbne data
