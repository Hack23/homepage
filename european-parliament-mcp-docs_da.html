Projektoversigt

European Parliament MCP Server er en TypeScript/Node.js-implementering af Model Context Protocol (MCP), der er udviklet til at give AI-assistenter struktureret, programmatisk adgang til Europa-Parlamentets åbne datasæt. Serveren forbinder til den officielle European Parliament Open Data API og stiller 62 specialbyggede værktøjer til rådighed til forespørgsel om MEP'er, plenarmøder, afstemningsregistre, udvalg og parlamentariske spørgsmål.

Projektet opnår SLSA Level 3-supply-chain-sikkerhed, opretholder 80%+ testdækning med 1.130+ enhedstests og 71 E2E-tests og indeholder omfattende ISMS-dokumentation i overensstemmelse med ISO 27001:2022, NIST CSF 2.0, CIS Controls v8.1 og GDPR.
