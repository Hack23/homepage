Accessibility Fonctionnalités

  • Semantic HTML: Uses <nav> with aria-label="Fil d'Ariane" and <ol> list structure
  • Attributs ARIA : Page actuelle marquée avec aria-current="page"
  • Navigation au Clavier : Tous les liens sont accessibles par tabulation avec indicateurs de focus visibles
  • Compatible Lecteur d'Écran : Structure sémantique appropriée annonce le contexte de navigation
  • Contraste de Couleur : Conforme aux normes WCAG 2.1 AA (4.5:1 minimum)
  • Design Responsive : S'adapte aux viewports mobile, tablette et bureau
  • Support de Thèmes : Fonctionne avec les modes clair, sombre et le thème Black Trigram
