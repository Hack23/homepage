🏛 EU Parliament Monitor

European Parliament Intelligence Platform

Open-source European Parliament Intelligence Platform monitoring political activity at the EU level with systematic transparency. Comprehensive tracking of MEPs, plenary sessions, committees, legislative documents, and voting records using official European Parliament open data.

📂 View on GitHub ✨ Features 📚 Documentation

🎯 Key Features

👤 MEP Monitoring

Current and historical information on Members of the European Parliament. Track voting behavior, committee memberships, legislative activity, and attendance records for all MEPs.

🖹️ Plenary Tracking

Comprehensive monitoring of plenary sessions including votes, attendance, speeches, and debates. Detailed analysis of EU legislative decisions and parliamentary proceedings.

🏛 Committee Monitoring

Track committee meetings, documents, and membership across all European Parliament committees. Monitor legislative preparation and committee-level political activity.

📄 Document Search

Powerful search and access to legislative texts, reports, amendments, and official documents. Navigate the full corpus of European Parliament documentation with ease.

❓ Parliamentary Questions

Track parliamentary questions and answers from MEPs. Monitor how representatives hold the European Commission and Council accountable through formal questioning procedures.

🤖 AI Access via MCP

Integration with the European Parliament MCP Server for AI-powered access to EU Parliament data. Enables AI assistants to query and analyze EP data programmatically.

🔧 Technology Stack

🌐 Frontend

HTML5, CSS3, JavaScript - Modern static web application with responsive design, accessible interface, and progressive enhancement.

📡 Data Source

European Parliament Open Data - Official data from data.europarl.europa.eu. Automated updates from authoritative EU institutional sources.

🤖 AI Integration

MCP Server Protocol - European Parliament MCP Server enables AI assistants to query and analyze EP data. Integrates with Claude, GPT, and other AI systems.

🔒 Security

ISMS Compliant - ISO 27001:2022, NIST CSF 2.0, CIS Controls v8.1, and GDPR compliant. Fully transparent security practices with public ISMS documentation.

🛡️ Security & Quality

EU Parliament Monitor demonstrates enterprise security practices for handling EU political data with integrity and transparency.
