🔍 Citizen Intelligence Agency

Politisk åpenhet gjennom åpne data

Åpen kildeetterretning (OSINT) plattform for politisk åpenhet i Sverige. Overvåk parlamentarisk aktivitet, avstemningsprotokoll og politisk oppførsel med omfattende datadrevne innsikter og ansvarsmålinger.

📂 Se på GitHub ✨ Funksjoner 📚 Dokumentasjon 🗳️ Valg 2026

🎯 Nøkkelfunksjoner

🏛️ Parlamentarisk overvåking

Sanntidssporing av Riksdagens aktivitet inkludert debatter, forslag, komitéarbeid og lovgivningsprosesser. Omfattende historisk dataanalyse.

🗳️ Analyse av avstemningsprotokoll

Detaljert analyse av alle parlamentariske avstemninger med sporing av partitilhørighet, individuelle avstemningsmønstre og koalisjonsanalyse. Åpenhet i demokratisk beslutningstaking.

📊 Politisk analyse

Datadrevne innsikter i politisk atferd, partistrategier og lovgivningstrender. Statistisk analyse av politisk ytelse og ansvarlighet.

📈 Datavisualisering

Interaktive dashboards og diagrammer for utforsking av politiske data. Visuell fremstilling av avstemningsmønstre, fremmøteregistreringer og lovgivningsaktivitet.

✅ Ansvarsmålinger

Spor politikernes løfter, oppmøte, avstemningskonsistens og aktivitetsnivå. Objektive målinger av politisk ytelse og engasjement.

🔓 Integrasjon av åpne data

Integrasjon med offisielle svenske regjeringens åpne datakilder. Automatiske oppdateringer og sanntidssynkronisering med parlamentariske systemer.

🎯 Misjon og innvirkning

🌟 Demokratisk åpenhet

Gir borgere tilgang til politisk informasjon og ansvarlighetsdata. Gjør demokratiet mer transparent og tilgjengelig for alle.

Virkning: Økt borgerengasjement i demokratiske prosesser gjennom bedre tilgang til politisk informasjon.

📰 Presseanerkjennelse

Omtalt i Computer Sweden og Riksdag och Departement for innovativ tilnærming til politisk åpenhet. Anerkjent av National Democratic Institute.

Virkning: Økt bevissthet om viktigheten av parlamentarisk overvåking og borgerteknologi.

🔍 Evidensbasert politikk

Gir objektive data til forskere, journalister og borgere for å analysere politisk atferd og holde representanter ansvarlige.

Virkning: Mer informert offentlig diskurs basert på fakta fremfor retorikk.

⚙️ Plattformkapabiliteter

👤 Politikerprofiler

Omfattende profiler for alle Riksdag-medlemmer inkludert avstemningshistorikk, partitilhørighet, komitémedlemskap og aktivitetsmålinger.

🏛️ Partianalyse

Statistikk på partinivå og koalisjonsanalyse. Spor partisamhold, avstemningsdisiplin og strategisk posisjonering på tvers av saker.

📋 Lovgivningssporing

Følg lovforslag og forslag gjennom lovgivningsprosessen. Spor endringsforslag, komitégjennomganger og endelige avstemninger med full historisk kontekst.

🔎 Søk og oppdagelse

Kraftig søkefunksjonalitet på tvers av debatter, avstemninger og dokumenter. Filtrer og analyser data etter emne, dato, politiker eller parti.

📊 Tilpassede rapporter

Generer tilpassede rapporter for bestemte tidsperioder, politikere eller saker. Eksporter data for forskning og analyse.

🔔 Varsler og meldinger

Hold deg informert om spesifikke politikere, partier eller saker. Automatiske varsler for avstemningsaktivitet og lovgivningsendringer.

🛠️ Teknologistakk

☕ Backend

Java, Spring Framework - Bedriftsklasse backend med robust databehandling, API-integrasjon og forretningslogikk.

🖥️ Frontend

Vaadin Framework - Moderne web-UI-rammeverk for rike, responsive brukergrensesnitt med serverside-arkitektur.

🗄️ Database

PostgreSQL - Pålitelig relasjonsdatabase for komplekse politiske dataforhold og historiske poster.

🔒 Security

SLSA Level 3 - Forsyningskjedesikkerhet, sikre utviklingspraksiser og kontinuerlig sikkerhetsovervåking.

🛡️ Sikkerhet og kvalitet

Citizen Intelligence Agency demonstrerer bedriftssikkerhetspraksiser for håndtering av sensitiv politisk data med integritet og tilgjengelighet.

💼 Bruksområder

👥 Borgere og velgere

Ta informerte valg basert på faktisk politisk atferd, ikke bare valgkampløfter. Spor aktiviteten til dine representanter.

📰 Journalister og forskere

Tilgang til omfattende politiske data for undersøkende journalistikk og akademisk forskning. Faktabasert rapportering om politisk atferd.

🏛️ Sivilsamfunnsorganisasjoner

Overvåk spesifikke politikkområder og hold politikere ansvarlige. Spor lovgivningsfremdrift i saker som er viktige for din organisasjon.

🎓 Lærere og studenter

Undervisningsverktøy for statsvitenskap, samfunnsfagopplæring og demokratiske prosesser. Virkelige data for akademiske studier.
