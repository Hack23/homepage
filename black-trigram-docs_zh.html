📖 游戏 개요

黑卦(Black Trigram)는 韩国 武术 철학과 주역에서 영감을 받은 현실적인 2D 정밀 格斗 游戏입니다. 이 游戏은 정밀한 해부학적 타겟팅을 통해 진정한 韩国 무술을 가르치며, 70개의 要害点점과 5개의 독특한 전투 原型을 현대적인 웹 技术로 구현했습니다.

🌐 웹 版本 (권장)

즉시 접근

웹 브라우저를 통해 시작하는 것이 가장 쉬운 방법입니다:

1. https://blacktrigram.com 방문 2. "지금 플레이" 클릭 3. 즉시 格斗 훈련 시작

🔧 로컬 开发

开发자 설정

开发 또는 로컬 호스팅을 위해:

# 저장소 복제 git clone https://github.com/Hack23/blacktrigram.git cd blacktrigram # 의존성 설치 npm install # 开发 서버 시작 npm run dev # 프로덕션용 빌드 npm run build # 테스트 실행 npm run test npm run test:combat

🎯 格斗 철학

"어둠 속에서 완벽한 일격을 찾아라"

"In darkness, seek the perfect strike"

각 기법의 초점:

  • 정확한 타격 (Precise Targeting) - 정확한 해부학적 취약점
  • 최대 효과 (Maximum Effectiveness) - 일격 무력화
  • 전투 심리 (Combat Psychology) - 格斗를 위한 정신적 준비
  • 传统 지식 (Traditional Knowledge) - 진정한 韩国 무술
  • 原型 특화 (Archetype Specialization) - 전투 유형별 독특한 접근

韩国 武术 기초

黑卦는 진정한 韩国 武术 원리를 기반으로 구축되어 传统 기법과 철학을 集成합니다:

🥋 跆拳 (Taekkyeon) - 传统 韩国 발차기

리듬, 균형, 연속적인 움직임을 강조하는 유동적인 다리 기법으로 格斗 효과를 극대화합니다.

🤼 씨름 (Ssireum) - 韩国 레슬링

던지기, 테이크다운, 그라운드 컨트롤 방법에 초점을 맞춘 그래플링 기법입니다.

🗡️ 검술 (Geomsul) - 검 武术

맨손 格斗 응용을 위해 적응된 传统 韩国 검술 원리입니다.

🎯 要害点술 (Geupso-sul) - 要害点점 타격

최대 格斗 효율을 위한 해부학적 취약점의 정밀한 타겟팅입니다.

⚔️ 비수 (Bisu) - 치명적 기법

최대 효과를 위한 결정적인 맨손 格斗 방법입니다.

格斗 메커니즘

黑卦는 진정한 武术 시뮬레이션을 위해 현실적인 格斗 物理학과 신체 역학을 구현합니다:

🩸 진정한 외상 系统

실제 格斗 손상 模式을 기반으로 한 현실적인 부상 시각화 및 출혈 효과입니다.

🦴 뼈 충격 오디오

현실적인 格斗 피드백을 위한 진짜 뼈 접촉 및 골절 소리입니다.

🫁 호흡 방해

호흡계 타겟팅이 체력과 格斗 성능에 영향을 미칩니다.

⚖️ 균형 系统

格斗 안정성에 영향을 미치는 현실적인 자세와 운동량 物理학입니다.

🧠 의식 상태

성공적인 格斗 기법으로부터의 점진적인 인식 손상입니다.

훈련 모듈

黑卦는 韩国 武术 기법을 마스터하기 위한 포괄적인 훈련 모듈을 제공합니다:

🎯 해부학 연구 (Anatomical Study)

70개의 해부학적 타겟 포인트, 정밀 타격, 고급 格斗 기법을 학습합니다.

⚔️ 무술 기법 (Martial Techniques)

기본 기법, 팔괘 武术, 콤비네이션 시퀀스를 마스터합니다.

🥊 실전 훈련 (Combat Training)

일대일 格斗, 환경 格斗, 다수 상대 시나리오를 연습합니다.

🎭 原型 특화 (Archetype Mastery)

5가지 格斗 原型 각각과 그들의 독특한 기법에 대한 전문 훈련입니다.

技术 架构

黑卦는 성능과 현실감을 위해 최적화된 현대적인 웹 技术로 구축되었습니다:

🎮 格斗 物理 엔진

엄격한 타입 안전성과 성능을 위해 PixiJS 8.x, React 19, TypeScript로 구축되었습니다.

⚡ 성능 최적화

빠른 开发을 위한 Vite, 하드웨어 가속을 위한 WebGL, 오디오를 위한 Howler.js입니다.

🎨 시각 효과

부드러운 格斗를 위한 맞춤 파티클 엔진, WebGL 셰이더, 60fps 애니메이션입니다.

🎯 格斗 컴포넌트

VitalPointTargeter, CombatTracker, TechniqueCalculator, CombatAnalyzer 系统입니다.
