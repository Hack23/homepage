📖 游戏 个

黑卦(Black Trigram) 韩国 武术 철학和 주역and 영감 받 현실的인 2D 精密 格斗 游戏. 游戏 精密的해부학的 타겟팅 통해 正宗的 韩国 武术 르, 70个的 要害点点和 5个的 독特的战斗 原型 现代的인 웹 技术 구현했습니다.

🚀 빠른 시작

🌐 웹 版本 (권장)

즉시 方法

웹 브라우저 통해 시작 것 장 쉬运 방법:

1. https://blacktrigram.com 访问 2. "금 플레" 클릭 3. 즉시 格斗 훈练 시작

🔧 컬 开发

开发자 설정

开发 也 컬 호스팅 为了:

# 저장소 克隆 git clone https://github.com/Hack23/blacktrigram.git cd blacktrigram # 的존性 설 npm install # 开发 and버 시작 npm run dev # 프덕션用 빌드 npm run build # 테스트 실행 npm run test npm run test:combat
