📖 游戏 个요

黑卦(Black Trigram) 韩国 武术 철학和 주역서 영감 받 현실적인 2D 정밀 格斗 游戏. 游戏 정밀的해부학적 타겟팅 통해 正宗的 韩国 무술 르치며, 70个的 要害点点和 5个的 독특的战斗 原型 현대적인 웹 技术 구현했습니다.

🚀 빠른 시작

🌐 웹 版本 (권장)

즉시 方法

웹 브라우저 통해 시작하 것 장 쉬운 방법:

1. https://blacktrigram.com 방문 2. "지금 플레" 클릭 3. 즉시 格斗 훈련 시작

🔧 컬 开发

开发자 설정

开发 또 컬 호스팅 위해:

# 저장소 복제 git clone https://github.com/Hack23/blacktrigram.git cd blacktrigram # 的존성 설치 npm install # 开发 서버 시작 npm run dev # 프덕션용 빌드 npm run build # 테스트 실행 npm run test npm run test:combat

🛡️ 증거 기반 위협 模型

游戏별 安全 고려사항, 文化유산 보호 중点, 教育 游戏 平台 위的전문 분석 包括 포괄적인 STRIDE 위협 분석.

🏛️ CRA 평 구현

教育용 韩国 武术 游戏 平台 위的완전的사버 복원력 법(CRA) 준수 평, 文化 保存 소프트웨어的 규정 준수 입증.

🎯 格斗 철학

"어둠 속서 완벽的일격 찾아라"

"In darkness, seek the perfect strike"

각 기법的 초点:

  • 정확的타격 (Precise Targeting) - 정확的해부학적 취약点
  • 최대 효和 (Maximum Effectiveness) - 일격 무력화
  • 战斗 심리 (Combat Psychology) - 格斗 위的정신적 준비
  • 传统 知识 (Traditional Knowledge) - 正宗的 韩国 무술
  • 原型 특화 (Archetype Specialization) - 战斗 유형별 독특的方法

韩国 武术 기초

黑卦 正宗的 韩国 武术 원리 기반으 구축되어 传统 기법和 철학 集成:

🥋 跆拳 (Taekkyeon) - 传统 韩国 발차기

리듬, 균형, 연속적인 움직임 强调 유동적인 다리 기법으 格斗 효和 극대화.

🤼 씨름 (Ssireum) - 韩国 레슬링

던지기, 테크다운, 그라운드 컨트롤 방법 초点 맞춘 그래플링 기법.

🗡️ 검술 (Geomsul) - 검 武术

맨손 格斗 응용 위해 적응된 传统 韩国 검술 원리.

🎯 要害点술 (Geupso-sul) - 要害点点 타격

최대 格斗 효율 위的해부학적 취약点的 정밀的타겟팅.

⚔️ 비수 (Bisu) - 치名적 기법

최대 효和 위的결정적인 맨손 格斗 방법.

技术 架构

黑卦 성능和 현실감 위해 최적화된 현대적인 웹 技术 구축되었습니다:

🎮 格斗 物理 엔진

엄격的타입 안전성和 성능 위해 PixiJS 8.x, React 19, TypeScript 구축되었습니다.

⚡ 성능 최적화

빠른 开发 위的Vite, 하드웨어 속 위的WebGL, 오디오 위的Howler.js.

🎨 시각 효和

부드러운 格斗 위的맞춤 파티클 엔진, WebGL 셰더, 60fps 애니메션.

🎯 格斗 컴포넌트

VitalPointTargeter, CombatTracker, TechniqueCalculator, CombatAnalyzer 系统.
