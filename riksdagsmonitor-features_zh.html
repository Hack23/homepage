人工智能驱动的瑞典议会情报

Riksdagsmonitor是世界首个为瑞典议会打造的全自主人工智能政治新闻编辑室。11个自主代理工作流程凭借91项技能生成深度政治分析——不是肤浅摘要，而是涵盖349名议员、2494名历史政治家、350万+投票记录及10.9万+文件的结构化OSINT情报，支持14种语言。基于Citizen Intelligence Agency数据骨干构建。

📰 人工智能新闻资讯 📊 实时仪表板 📚 DeepWiki 📂 GitHub

🏛️ 议会监控

👤 议员追踪

为所有349名Riksdag议员提供完整档案。追踪个人投票历史、政党归属、委员会成员资格、质询记录、发言及随时间变化的活动指标。

🗳️ 投票记录

对每次议会投票进行完整投票记录分析。追踪政党立场、个人投票模式、联盟行为及跨立法届期的投票一致性。

🏢 委员会活动

监控所有Riksdag委员会，包括组成、会议频率、提案处理及政策影响。追踪委员会级别的决策模式和立法影响。

💬 质询

追踪向部长提出的所有质询（议会质问）。通过完整的质询记录监控响应率、响应时间及政府问责情况。

📜 立法提案

跟踪法案、动议和提案在整个立法程序中的进展。在完整历史背景下追踪修正案、委员会审查、全体讨论及最终投票。

🏛️ 政党分析

政党级统计与跨党联盟分析。追踪政党凝聚力、投票纪律、议会团体动态及跨议题战略定位。

📅 历史数据

📊 50年以上档案

涵盖50余年瑞典议会活动的完整历史档案。分析从1970年代至今的长期政治趋势、政策演变及民主发展。

📈 趋势分析

识别投票行为、政党定位及政策优先事项的长期模式。将当前政治气候与历史基准进行比较，以了解系统性变化。

🗓️ 立法届期比较

比较不同Riksdag届次和政府组成期间的政治活动。分析政治权力转移如何影响立法产出和民主问责。

🌍 14种语言支持

🗣️ 国际可访问性

完整的14种语言平台支持：瑞典语、英语、阿拉伯语、中文、丹麦语、荷兰语、芬兰语、法语、德语、希伯来语、日语、韩语、挪威语和西班牙语。让瑞典民主面向全球开放。

🔤 本地化界面

完整的界面本地化，包括对阿拉伯语和希伯来语的从右到左（RTL）支持。所有导航、标签和描述性内容均针对每种支持的语言进行了完整翻译。

🌐 WCAG 2.1 AA无障碍

所有语言版本均完全符合WCAG 2.1 AA标准。适合残障用户使用，屏幕阅读器兼容，全程键盘可导航。

📊 政治分析

🧮 行为分析

先进的政治行为分析，衡量竞选承诺与实际投票记录之间的一致性。识别政治决策模式和问责差距。

📉 透明度指标

针对个人议员、政党及政府机构的客观透明度指标。根据出席率、参与度、响应率及立法参与度对政治行为者进行评分。

📊 数据可视化

用于探索议会数据的交互式仪表板和图表。投票模式、出席记录、立法活动及政治趋势的可视化呈现。

🔎 高级搜索

跨所有辩论、投票、质询和文件的强大搜索功能。按主题、日期范围、政治家、政党或委员会筛选和分析数据。

📑 自定义报告

针对特定时间段、政治家或政策领域生成自定义报告。导出数据用于研究、新闻和学术分析。

🔗 API访问

提供编程访问议会数据的完整API文档。完整文档见riksdagsmonitor.com/docs/api/。

✅ 数据质量与完整性

🔍 自动化验证

针对官方Riksdag开放数据源的持续自动化数据验证。检测并标记不一致之处，确保整个管道中的数据完整性。

🔐 数字签名

通过数字签名验证数据完整性。所有数据直接来源于瑞典Riksdag官方API，确保真实性并防止篡改。

📡 官方数据源

专属来源于瑞典Riksdag官方开放数据系统。无人工录入，无偏差——与权威政府数据API直接集成。

🧪 全面测试

完整测试覆盖，包括单元测试、端到端系统测试及性能基准测试。测试结果和覆盖率报告见riksdagsmonitor.com/docs/coverage/。

完全符合ISO 27001:2022信息安全管理标准。完整的安全控制文档在GitHub上公开发布。
