⚡ Combat Mastery

Black Trigram is a realistic combat simulator that teaches authentic Korean martial arts through precise anatomical targeting. Master traditional vital-point techniques via modern 2D combat mechanics across 5 distinct fighter archetypes.

A realistic 2D precision combat game inspired by Korean martial arts philosophy and the I Ching. Built with modern web technologies including PixiJS, React 19, and TypeScript for optimal performance and authentic combat simulation.

Special Focus: Military discipline, overwhelming force

🥷 암살자 (Amsalja) - Shadow Assassin

Combat Philosophy: Efficiency through invisibility

Special Focus: Stealth approaches, instant takedowns

💻 해커 (Hacker) - Cyber Warrior

Combat Philosophy: Information as power

Special Focus: Environmental manipulation, tech-assisted strikes

🕵️ 정보요원 (Jeongbo Yowon) - Intelligence Operative

Combat Philosophy: Knowledge through observation

Special Focus: Psychological manipulation, precise timing

⚡ 조직폭력배 (Jojik Pokryeokbae) - Organized Crime

Combat Philosophy: Survival through ruthlessness

Special Focus: Dirty fighting, improvised weapons
