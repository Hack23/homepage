Project Oversikt

The CIA Compliance Manager is a comprehensive web application designed to help organizations assess, implement, og manage security controls across the CIA triad (Konfidensialitet, Integritet, Tilgjengelighet). The application enables users to evaluate security posture, measure compliance against major frameworks, og analyze the business impacts of security implementations.
