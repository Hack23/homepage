Projektoversigt

EU Parliament Monitor er en omfattende open source-platform til overvågning af Europa-Parlamentets aktiviteter med systematisk transparens. Platformen følger MEP'er, plenarmøder, udvalg, lovgivningsdokumenter og afstemningsregistre ved hjælp af officielle åbne data fra Europa-Parlamentet. Platformen er fuldt ISMS-kompatibel med ISO 27001:2022, NIST CSF 2.0, CIS Controls v8.1 og GDPR.

📖 MCP API Docs 📈 Coverage 📚 DeepWiki

Nøglefunktioner

MEP-overvågning

Aktuel og historisk MEP-information inkl. stemmehistorik, udvalgsposter, fremmøde og parlamentarisk aktivitet.

Plenarsporning

Omfattende sporning af plenarmøder inkl. stemmer, fremmøde, taler og debatter.

Udvalgsovervågning

Overvåg udvalgsmøder, dokumenter, ordførere og medlemskaber på tværs af alle Europa-Parlamentets udvalg.

Dokumentsøgning

Søg og tilgå lovtekster, betænkninger, ændringsforslag og officielle EU-parlamentsdokumenter.

Parlamentariske spørgsmål

Spor parlamentariske spørgsmål og institutionelle svar. Overvåg MEP-ansvarlighed.

MCP-serverintegration

Integration med European Parliament MCP Server for AI-drevet adgang til EP-data via Model Context Protocol.
