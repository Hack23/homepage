תיעוד Black Trigram

ארכיטקטורה טכנית, הטמעה, ואבטחה

📚 סקירה כללית של התיעוד

תיעוד מקיף עבור Black Trigram, סימולטור קרב חינוכי של אומנויות לחימה קוריאניות. כולל מידע טכני למפתחים, מעצבים משחקים, ומשתמשים.

🏗️ ארכיטקטורה C4

Black Trigram משתמש במודל ארכיטקטורה C4 (Context, Containers, Components, Code) לתיעוד ברור ומובנה:

  • Context (הקשר): סקירה כללית של המשחק במערכת אקולוגית רחבה יותר
  • Containers (מכולות): Unity game engine, מנוע פיזיקה, מערכת קרב
  • Components (רכיבים): מנהל נקודות חיוניות, מערכת ארכיטיפים, מנהל I Ching
  • Code (קוד): פרטי יישום כיתות ושיטות

🎮 אינטגרציה Unity

המשחק נבנה עם Unity 2021.3 LTS ומשתמש בדפוסי עיצוב מודרניים:

  • ECS (Entity Component System): ביצועים מיטביים
  • PhysX: מנוע פיזיקה מדויק למכות אמיתיות
  • ScriptableObjects: ניהול נתונים מודולרי
  • State Machine: ניהול מצבי לוחם ואנימציות

⚔️ עיצוב מערכת הקרב

מערכת קרב מתוחכמת המדמה טכניקות אומנויות לחימה אמיתיות:

  • מערכת 70 נקודות חיוניות: מיקום אנטומי מדויק
  • חישוב נזק מבוסס פיזיקה: כוח, זווית, מסה
  • 5 ארכיטיפים ייחודיים: סגנונות קרב שונים
  • אינטגרציה I Ching: עיתוי אסטרטגי ותזמון

🛡️ אבטחה והתמקדות בשרשרת האספקה

Black Trigram מיישם אבטחה ברמה תעשייתית:

  • SLSA Level 3: הקשחה מלאה של שרשרת האספקה
  • OpenSSF Scorecard: ניטור ביטחון מתמשך
  • קוד מבוקר: סקירת עמיתים לכל PR
  • בנייה אוטומטית: CI/CD עם GitHub Actions
