🏛️ EU Parliament Monitor

人工智能驱动的欧洲议会情报

欧洲议会专属的开源政治透明度平台。8个统一gh-aw工作流程以14种语言生成自主政治新闻——51件分析基准、17种方法论、52个模板。每日170+次运行，产出1700+件成果。基于European Parliament MCP Server（62个工具）驱动。

🌐 访问EU Parliament Monitor 📂 在GitHub上查看 ✨ 功能特性 📚 技术文档

🎯 核心功能

🤖 8个统一人工智能工作流程

8个统一gh-aw工作流程（另加1个翻译流程）产出51件分析基准，涵盖17种方法论和52个模板。每日170+次运行，自主生成1700+件分析成果，零人工干预。

🔌 62个MCP工具驱动

由European Parliament MCP Server提供支持，包含62个MCP工具（15个OSINT情报工具+47个数据访问工具），为人工智能工作流程提供实时欧洲议会数据。

🌍 14种语言新闻生成

以14种语言自主生成欧洲议会新闻和分析，让更广泛的公民受众能够了解欧盟立法进程，促进民主参与。

📊 欧洲议员与全会追踪

全面监控欧洲议员活动、全会表决及出席记录。追踪政党立场、联盟动态及跨立法届期的个人问责情况。

🏛️ 委员会情报

深度监控欧洲议会委员会活动——会议记录、文件、报告员追踪，以及立法程序在委员会层面的进展情况。

📄 立法文件搜索

强大的搜索功能，覆盖欧洲议会立法文件、修正案、报告及官方文件的完整语料库，提供多语言访问支持。

🎯 平台概览

🌟 欧洲议会透明度

EU Parliament Monitor专注于欧盟层面的政治透明度，通过欧洲议会官方开放数据，使公民、研究人员、记者和公民社会组织能够通过数据驱动的洞察追究欧盟代表的责任。

🔄 人工智能新闻生成

人工智能工作流程自动运行，持续生成议员档案、委员会报告、全会分析及专项政策追踪内容，无需编辑团队参与。

🔍 开放数据集成

专属来源于欧洲议会官方开放数据API，并对所有数据来源进行完整验证。数据完整性通过自动化测试套件保障。

💼 使用场景

👥 欧盟公民

以母语了解欧洲议会活动。追踪您所在国家的欧洲议员行为，并就欧盟立法程序做出明智判断。

📰 记者与研究人员

获取全面的欧洲议会数据，用于调查性新闻报道和学术研究。快速识别投票模式和欧盟政策趋势。

🏛️ 公民社会与倡导组织

监控特定欧盟政策领域并追究议员责任。追踪与贵组织优先议题相关的立法进展。

🤖 人工智能与开发者

通过European Parliament MCP Server，您的人工智能助手可以直接查询欧洲议会数据，实现欧盟政治情报在工作流程中的集成。
