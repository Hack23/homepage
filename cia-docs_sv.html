Citizen Intelligence Agency Dokumentation senaste

Omfattande dokumentation för Citizen Intelligence Agency - en oberoende, volontärdriven OSINT-plattform för riksdag, regering, myndigheter och globala indikatorer i Sverige.

Projektöversikt

Citizen Intelligence Agency är ett volontärdrivet projekt för öppen källkodsunderrättelse (OSINT) som tillhandahåller omfattande analys av politiska aktiviteter i Sverige. Plattformen täcker fyra huvudområden:

  • Riksdagen - Sveriges parlament med ledamöter, utskott och beslutsprocesser
  • Regeringen - Ministerier och ministeriell verksamhet
  • Myndigheter - Statliga verk och administrativa organ
  • Världsbanken Indikatorer - Globala ekonomiska och utvecklingsindikatorer

Genom avancerad övervakning av viktiga politiska personer och institutioner levererar plattformen:

  • Finansiella prestandamått
  • Riskbedömningsanalys
  • Politisk trendanalys
  • Politikerrangsystem
  • Prestandajämförelser
  • Transparensinsikter

Vårt initiativ förblir strikt oberoende och opartiskt, med fokus på att främja informerat beslutsfattande och öka demokratiskt engagemang.

Nyckelfunktioner

Citizen Intelligence Agency tillhandahåller en omfattande uppsättning funktioner för politisk transparens och analys:

📊 Interaktiva Dashboards

Realtids-dashboards som visualiserar politiska aktiviteter, styrningsmetrik och institutionell prestanda över svenska politiska enheter.

🏆 Politiska Resultattavlesystem

Kvantitativa bedömningar och rankningar av politiska personer, partier och institutioner baserat på objektiva prestandamått.

📈 Kritiska Analysverktyg

Avancerade dataanalysförmågor som ger insikter i politiska trender, röstmönster och styrningseffektivitet.

🔍 Transparensmått

Standardiserade mätningar av transparens över statliga enheter, som belyser områden av styrka och möjligheter till förbättring.

⚖️ Ansvarsåtgärder

Verktyg för att spåra åtaganden, röstningskonsistens och överensstämmelse mellan politiska löften och policyimplementering.

📱 Datadrivna Insikter

Evidensbaserade bedömningar drivna av öppna datakällor för att tillhandahålla objektiva analyser av politisk prestanda.

Driftsättningsalternativ

AWS CloudFormation-driftsättning

Medborgarunderrättelsebyrån kan driftsättas på AWS med hjälp av vår tillhandahållna CloudFormation-mall:

  1. Ladda ner CloudFormation-stackfilen
  2. Skapa en ny stack i AWS CloudFormation-konsolen
  3. Ladda upp mallfilen och konfigurera parametrar
  4. Bekräfta IAM-resursskapande och starta stacken
  5. Få åtkomst till applikationen via URL:en i stackutdata

Debian/Ubuntu-installation

För lokal eller självhanterande driftsättning på Debian/Ubuntu 24.04+:

  1. Installera förutsättningar:
    sudo apt-get update && sudo apt-get install -y openjdk-21-jdk postgresql-16 postgresql-contrib postgresql-16-pgaudit
  2. Konfigurera PostgreSQL enligt detaljerna i PostgreSQL-konfigurationsguiden.
  3. Skapa databas och användare:
    sudo su - postgres
psql
postgres=# CREATE USER eris WITH password 'discord';
postgres=# CREATE DATABASE cia_dev;
postgres=# GRANT ALL PRIVILEGES ON DATABASE cia_dev to eris;
postgres=# \q
exit
  4. Ladda ner och installera det senaste Debian-paketet:
    # Ersätt 2025.1.2 med den senaste versionen om nödvändigt
VERSION="2025.1.2" 
wget https://github.com/Hack23/cia/releases/download/${VERSION}/cia-dist-deb-${VERSION}.all.deb
sudo dpkg -i cia-dist-deb-${VERSION}.all.deb
# Om det finns beroendeproblem, kör: sudo apt --fix-broken install
  5. Åtkomst till servern på https://localhost:28443/cia/

💼 Affärsfall för Nyhetsorganisationer

Översikt: CIA för Mediaintelligens

Citizen Intelligence Agency ger nyhetsorganisationer systematisk politisk övervakning i stor skala, vilket möjliggör datadriven journalistik som tidigare var resursprohibitiv. Med 45 beteenderiskregler över 5 analytiska ramverk levererar CIA omfattande täckning av 349 svenska politiker, 10 000+ årliga omröstningar och 20 000+ dokument. Plattformen kombinerar realtidsövervakning med 50 års historisk data från den svenska riksdagen, vilket ger oöverträffat djup för politisk analys. Vår beprövade metodik och transparens med öppen källkod säkerställer tillförlitlighet och förtroende—kritiska faktorer för nyhetsorganisationer som satsar sitt rykte på korrekt rapportering.

Underrättelseproduktportfölj

Politisk Underrättelse-API

RESTful API med OAuth 2.0-autentisering och 99,5% drifttids-SLA. Åtkomst till 349 politikerprofiler, 10 000+ årliga omröstningar, 20 000+ dokument genom standardiserade endpoints. Realtidsuppdateringar levererar röstningsresultat inom minuter efter parlamentariska beslut, medan dagliga riskbedömningar lyfter fram framväxande problem. Gränsbegränsningar skalar med vald nivå, och omfattande dokumentation säkerställer snabb integration i befintliga nyhetsrumsarbetsflöden.

Avancerad Analysplattform

Interaktiva dashboards designade för icke-tekniska användare gör det möjligt för journalister att utforska politisk data utan programmerings- kunskaper. Anpassad rapportgenerering stöder PDF-, Excel- och PowerPoint-format för redaktionsmöten och publicering. Automatiserad avisering via e-post, SMS och Slack säkerställer att möjligheter för breaking news aldrig missas. Enterprise-nivån inkluderar white-label-brandingalternativ, vilket gör det möjligt för nyhetsorganisationer att presentera CIA-intelligens under sin egen varumärkesidentitet.

Riskbedömningsflöde

45 beteenderegler analyserar politiker-, parti-, utskotts- och departementaktiviteter över fyra operativa domäner. Allvarlighetsgradklassificeringssystem (MINOR: 10-49, MAJOR: 50-99, CRITICAL: 100+) prioriterar undersökningsledtrådar. Historisk trendanalys avslöjar mönster över tid, medan prediktiv modellering förutspår koalitionsdynamik och policyskiften. Transparent metodik med alla 45 regler offentligt dokumenterade säkerställer att redaktionsteam förstår exakt vad som utlöser varningar, vilket stöder försvarbar journalistik.
