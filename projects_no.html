🚀 Åpen Kildekode Prosjekter

Demonstrerer sikkerhetsbest practices gjennom produksjonsapplikasjoner

Hos Hack23 snakker vi ikke bare om sikkerhet—vi beviser det gjennom åpen kildekode-prosjekter som demonstrerer virkelig implementering av sikkerhetskontroller, samsvarrammer og DevSecOps-praksis. Hvert prosjekt oppnår SLSA Level 3 forsyningskjedesikkerhet og opprettholder aktiv sikkerhetsovervåking.

🌟 Fremhevede Prosjekter

🥋 Black Trigram (흑괘)

Flaggskip Pedagogisk Spillprosjekt

Realistisk 3D presisjonskampsimulator inspirert av tradisjonelle koreanske kampsporter. Inneholder 70 anatomiske vitale punkter, 5 distinkte kjemperarketyper og autentiske kampteknikker med pedagogisk verdi og fokus på kulturell bevaring.

Nøkkelfunksjoner:

  • 70 Anatomiske Vitale Punkter
  • 5 Unike Kamparketype
  • Koreanske Kampsportsteknikker
  • Fysikkbasert Kamp
  • Pedagogisk Gameplay
  • Kulturell Autentisitet

🔐 CIA Compliance Manager

Bedriftssikkerhetsvurderingsplattform

Omfattende sikkerhetsvurderingsplattform for evaluering av Konfidensialitet, Integritet og Tilgjengelighet (CIA-triaden) med forretningspåvirkningsanalyse og samsvarmapping til NIST, ISO 27001, GDPR, HIPAA, SOC2 og CRA-rammeverk.

Nøkkelfunksjoner:

  • CIA-triade-vurdering
  • Forretningspåvirkningsanalyse
  • Multi-rammeverk-mapping
  • Trusselmodellering (STRIDE)
  • Bevisinnsamling
  • Samsvarrapportering

🔍 Citizen Intelligence Agency

Politisk Gjennomsiktighetsplattform

Åpen kildekode etterretningsplattform (OSINT) for politisk gjennomsiktighet i Sverige. Overvåker parlamentarisk aktivitet, stemmeregistre og politisk atferd med datadrevne innsikter, analyse-dashboards og ansvarlighetsmålinger.

Nøkkelfunksjoner:

  • Parlamentsovervåking
  • Stemmeregistranalyse
  • Politisk Analyse
  • Datavisualisering
  • Ansvarlighetsmålinger
  • Åpen Data-integrasjon

🏛️ EU Parliament Monitor

Etterretningsplattform for Europaparlamentet

Åpen kildekode Europaparlamentet etterretningsplattform som overvåker politisk aktivitet på EU-nivå med systematisk åpenhet. Omfattende sporing av MEP-er, plenumssesjoner, komiteer, lovgivningsdokumenter og avstemningsregistre.

Nøkkelfunksjoner:

  • MEP-overvåking
  • Sporing av plenumssesjoner
  • Komitéovervåking
  • Søk i lovgivningsdokumenter
  • Analyse av avstemningsregistre
  • MCP Server KI-integrasjon

🏛️ European Parliament MCP Server

KI-Drevet Parlamentarisk Datatilgang

Model Context Protocol (MCP) server som gir KI-assistenter strukturert tilgang til Europaparlamentets åpne datasett. Gjør det mulig for KI-verktøy å spørre om MEP-data, plenarsessjoner, komiteer, avstemninger og lovgivningsdokumenter i sanntid.

Nøkkelfunksjoner:

  • MCP-protokollintegrasjon
  • MEP-datatilgang
  • Plenarsesjonsspørringer
  • Komitéinformasjon
  • Søk i lovgivningsdokumenter
  • EU-parlamentsdata i sanntid

🗳️ Riksdagsmonitor

Etterretningsplattform for Sveriges Riksdag

Swedish Parliament monitoring platform som sporer lovgivere, avstemningsrekorder og parlamentarisk aktivitet. Gir datadrevne innsikter i svenske politiske prosesser med omfattende analyser.

Nøkkelfunksjoner:

  • Overvåking av Sveriges Riksdag
  • Lovgiveroppfølging
  • Analyse av avstemningsrekorder
  • Politisk analyse
  • Ansvarsmetrikker
  • Integrasjon med åpne data
