Projektin yleiskatsaus

EU Parliament Monitor on kattava avoimen lähdekoodin alusta Euroopan parlamentin toiminnan seurantaan systemaattisella läpinäkyvyydellä. Se seuraa MEP:ejä, täysistuntoja, valiokuntia, lainsäädäntöasiakirjoja ja äänestystuloksia virallisilla Euroopan parlamentin avoimilla datanlähteillä. Alusta on täysin ISMS-vaatimustenmukainen: ISO 27001:2022, NIST CSF 2.0, CIS Controls v8.1 ja GDPR.

Keskeiset ominaisuudet

MEP-seuranta

Nykyiset ja historialliset MEP-tiedot, mukaan lukien äänestyshistoria, valiokunnan jäsenyydet, läsnäolo ja parlamentaarinen aktiivisuus.

Täysistuntoseuranta

Kattava täysistuntojen seuranta, mukaan lukien äänestykset, läsnäolo, puheenvuorot ja väittelyt.

Valiokuntaseuranta

Seuraa valiokunnan kokouksia, asiakirjoja, esittelijöitä ja jäsenyyttä kaikissa Euroopan parlamentin valiokunnissa.

Asiakirjahaku

Hae ja käytä lainsäädäntötekstejä, raportteja, muutoksia ja virallisia EU-parlamentin asiakirjoja.

Parlamentaariset kysymykset

Seuraa parlamentaarisia kysymyksiä ja institutionaalisia vastauksia. Seuraa MEP:ien vastuullisuustoimia.

MCP Server -integraatio

Integraatio European Parliament MCP Server -palvelimen kanssa tekoälypohjaista EP-datankäyttöä varten Model Context Protocol -protokollan kautta.
